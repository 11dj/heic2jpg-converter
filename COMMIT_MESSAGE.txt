Title:
------
Add platform-specific HEIC conversion for macOS and Windows

Body:
-----
This commit implements platform-specific HEIC conversion:

1. macOS Support (using native sips command):
   - Get image dimensions: sips -g pixelWidth -g pixelHeight
   - Generate thumbnails: sips -Z size -s format jpeg
   - Convert HEIC to JPEG: sips -s format jpeg -s formatOptions quality
   - No external dependencies needed

2. Windows Support (using PowerShell + .NET Imaging):
   - Get image dimensions: System.Drawing.Image.FromFile
   - Generate thumbnails: System.Drawing.Bitmap resize
   - Convert HEIC to JPEG: ImageCodecInfo with quality encoder
   - Requires HEIF Image Extensions from Microsoft Store

3. Removed Linux support as requested
   - GitHub Actions only builds macOS and Windows now
   - Simplified workflow

4. Platform-specific compilation:
   - Uses #[cfg(target_os = "macos")] for macOS code
   - Uses #[cfg(target_os = "windows")] for Windows code
   - Clean separation of platform implementations

Platform Support:
- macOS: ✅ Full HEIC support (native sips)
- Windows: ✅ Full HEIC support (PowerShell/.NET)
- Linux: ❌ Removed

Prerequisites:
- macOS: No additional software needed
- Windows: Install "HEIF Image Extensions" from Microsoft Store
